# Ecuador, Are You Serious? 🇪🇨😂

A playful, **good-natured** single-page website that lovingly roasts the Ecuador
national soccer team — built as an inside joke for a friend who *will not stop*
bragging that Ecuador beat Germany. 🏆

It's all cheeky soccer banter between friends. No insults about the country or
its people — just teasing about one very memorable win.

## ✨ What's inside

- **Big bold headline** with rotating taunts that cycle every few seconds.
- **Official Excuse Generator™** — a button that spits out random funny reasons
  the win "didn't count."
- **Scoreboard of Shame** — absurd, totally-made-up stats that count up as you
  scroll.
- **Countdown timer** to the next big win… set to the year **2525**. ⏳
- **Confetti** that bursts whenever you click *anywhere* on the page. 🎉
- Mobile-responsive, themed in Ecuador's yellow / blue / red (used ironically).
- 100% vanilla **HTML / CSS / JS** — no frameworks, no build step.

## 📂 Files

| File         | Purpose                                            |
| ------------ | -------------------------------------------------- |
| `index.html` | Page structure and content                         |
| `style.css`  | All styling (Ecuador colors, layout, animations)   |
| `script.js`  | Taunts, excuse generator, countdown, confetti      |

## 🖥️ Run it locally

No build tools needed. Either:

- **Double-click `index.html`** to open it in your browser, or
- Serve it locally (recommended for the most accurate preview):

  ```bash
  # Python 3
  python3 -m http.server 8000
  # then visit http://localhost:8000
  ```

## 🚀 Deploy on GitHub Pages (step by step)

This repo is already on GitHub. To put the site live for free:

1. Push these files to the `main` branch (if you haven't already):

   ```bash
   git add .
   git commit -m "Add the website"
   git push origin main
   ```

2. On GitHub, open the repository and click **Settings** (top menu).
3. In the left sidebar, click **Pages**.
4. Under **Build and deployment → Source**, choose **Deploy from a branch**.
5. Set the branch to **`main`** and the folder to **`/ (root)`**, then click
   **Save**.
6. Wait ~1 minute. GitHub Pages will publish your site and show the live URL at
   the top of the **Pages** settings screen. It will look like:

   ```
   https://<your-username>.github.io/ecuador-are-you-serious/
   ```

7. Open that link, send it to your Ecuadorian friend, and enjoy. 😄

> 💡 Changes you push to `main` afterward will redeploy automatically within a
> minute or two.
